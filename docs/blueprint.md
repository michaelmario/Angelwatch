# **App Name**: AngelWatch

## Core Features:

- Secure User Authentication: Allow users (both clients and drivers) to securely register, log in, and manage their basic profiles. Data stored in Firestore.
- Real-time Driver Location Tracking: Drivers can share their real-time GPS location, which is securely updated and stored in Firestore to enable live tracking.
- Interactive Map Display for Clients: Clients can view available drivers and their current locations on an interactive map, dynamically updating as drivers move.
- Service Request & Assignment System: Clients can initiate service requests, which are then assigned to the nearest available driver. Request data is managed in Firestore.
- Intelligent Status Update Tool: Generate clear, concise, and personalized real-time status updates for clients, informing them about their service request's progress and estimated times, using AI to contextualize data.
- Progressive Web App (PWA) Installability: Enable the application to be installed on user devices, offering offline access and a native-like experience.

## Style Guidelines:

- Primary Color: A sophisticated, deep cadet blue (#2D598F) to evoke reliability and professionalism, forming the foundation of interactive elements and branding.
- Background Color: A very light, crisp off-white with a hint of blue (#EBF1F4) for the main background, providing a clean and open canvas that ensures excellent readability and a modern feel.
- Accent Color: A vibrant yet refined royal purple (#7566FF) to highlight call-to-action buttons, key interactive elements, and alerts, providing a modern contrast.
- Font Family: 'Inter' (sans-serif) for both headlines and body text. Its neutral, highly legible design ensures clarity and professionalism across all screen sizes.
- Use a set of clear, concise, and universally recognizable line icons for navigation, status indicators, and actionable elements, reinforcing the app's clean and functional design.
- Employ a responsive, card-based layout with generous whitespace to ensure information is easy to digest. Prioritize mobile-first design for a seamless PWA experience, ensuring optimal display of map and status updates.
- Incorporate subtle and quick animations for real-time map updates, new status notifications, and transitions between screens to enhance user feedback and improve the perception of responsiveness.
